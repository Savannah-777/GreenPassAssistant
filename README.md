# 🚚 绿通查验归档助手 (GreenPassAssistant)

这是一个针对“绿色通道”查验场景开发的自动化工具。它通过 **微信小程序** 拍照并智能重命名，结合 **Python 电脑端脚本** 自动监控归档，解决了查验视频文件名乱码、整理繁琐的问题。

## 🌟 核心功能

* **小程序端 (miniprogram)**：
    * **AI 实时识别**：集成百度 OCR，拍摄时自动锁定车牌号。
    * **自动命名**：视频录制完成后，自动保存为 `车牌-时间.mp4` 格式。
* **电脑端 (Python)**：
    * **多重路径解析**：自动扫描微信视频接收路径。
    * **智能截帧补录**：针对微信同步导致的乱码文件名，脚本会自动截取视频第一秒画面进行“二次识别”，确保归档准确。
    * **按日自动分类**：一键将所有视频按“年-月-日”分文件夹存放。
